## Arguments

### `Numbers.txt`

Write your phone number in ```905555555555``` Format 

### `Gonderildi.txt`
 
* Telephone Numbers Sent to Message (`If You Want to Send Messages Again to Same Numbers, Delete the Phone Numbers in the File`)
